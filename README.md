# Andrey's Personal Skills

This repository contains personal skills and workflows designed for AI agents (like Gemini or Claude) to help automate repetitive tasks.

## Skills Included:
- **[Telegram Sticker Manager](skills/telegram-sticker-manager/SKILL.md):** Automatically converts media to VP9 WebM and adds it to the 'Andrey Pack' Telegram sticker set.

## How to use:
Point an agent to the relevant skill directory. Each skill follows the standardized format for agent triggering and execution.
